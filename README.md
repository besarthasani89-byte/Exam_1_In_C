# Student Records (C Program)

This is a small C program to manage simple student records.

Each student has:
- An **ID** (Auto generated)
- A **NAME**
- An **AGE** (between 5 and 75 years)

The data is saved in a binary file called `Students.bin`.

---

## Features

- Show all students in the file
- Create a new student
- Show one student by ID
- Edit an existing student
- Simple text menu in the terminal
- Input checked for errors

---

## Build

Bash: gcc Exam.c -o Exam
